## Instalação

Clone o repositório utilizando o Git.

```bash
git clone 
```

Acesse a pasta do repositório e depois a dos exercicios exercicios
```bash
cd java-logic-exercise
```

```bash
cd src
cd exercises
```

Rode o exercicios que quiser
```bash
javac NOME_DO_EXERCICIO
```
